# Playwright Test Automation Project

## 🎯 Project Overview

This project contains automated end-to-end tests for the Sauce Demo e-commerce website (https://www.saucedemo.com/) using Playwright with TypeScript. The test suite covers various user scenarios including login flows, shopping cart operations, checkout processes, and visual regression testing.

## 🧪 Test Categories


### 1. **Basic Test Implementation** (`assessment-Task/`)
- Direct Playwright API usage without Page Object Model
- Covers fundamental e-commerce workflows
- Good for understanding basic Playwright concepts

### 2. **Page Object Model Tests** (`pom-Assessment/`)
- Implements design patterns for better maintainability
- Separates page logic from test logic
- Reusable components for common actions

### 3. **Visual Testing** (`visual-Testing-Scenarios-Assessment/`)
- Screenshot comparison tests
- Responsive design validation
- Visual regression testing

### 4. **Practice Tests** (`workersPractises/`)
- Additional test scenarios for practice
- Explores different testing approaches
